## Folders Structure
In the folder `app` is located the main code of the app
In the folder `test` is located the unit test

## How to install it
Execute
```bash
$ npm install
$ yarn install
```

either of the above to download dependencies

## How to test it
Execute:
```bash
$ npm test
$ yarn test
```
either of the above to test
